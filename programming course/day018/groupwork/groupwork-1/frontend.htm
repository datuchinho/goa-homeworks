Front-end დეველოპმენტი

სლაიდების დიაპაზონი: სლაიდები 3–53

მოკლე მიმოხილვა:

Front-end დეველოპმენტი ნიშნავს ვებსაიტის იმ ნაწილის შექმნას, რომელსაც მომხმარებელი ხედავს და გამოიყენებს. ამ კურსში ისწავლი როგორ შექმნა ვიზუალურად ლამაზი, ფუნქციური და მობილურებზე მორგებული ვებსაიტები HTML-ის, CSS-ის და JavaScript-ის გამოყენებით. შეისწავლი როგორ დაალაგო კონტენტი, როგორ დააკმაყოფილო დიზაინის წესები და როგორ გააკეთო საიტი ინტერაქტიული.

რას ისწავლი:

  • HTML (ჰიპერტექსტის მარკირების ენა): ვებსაიტების საწყისი სტრუქტურა. ისწავლი როგორ შექმნა სექციები, ბლოკები და სტრუქტურირებული გვერდები ელემენტების მეშვეობით, როგორიცაა <div> და <section>.
  • CSS (სტილის ფურცლები): ისწავლი როგორ გაალამაზო საიტი ფერებით, შრიფტებით, განლაგებით და მობილურებზე მორგებული დიზაინით (responsive design). გაიგებ რა არის box model, flexbox, grid და media queries.
  • JavaScript: პროგრამირების ენა, რომელიც საიტს ცოცხალს ხდის. შეისწავლი ცვლადებს, ფუნქციებს, მოვლენებს (events), ციკლებს, პირობას და როგორ მართო HTML ელემენტები კოდის მეშვეობით (DOM).

შეძენილი უნარები:

  • მობილურებზე მორგებული საიტების შექმნა
  • ინტერაქტიული ფუნქციების დამატება (ღილაკები, ფორმები, ანიმაციები და ა.შ.)
  • დიზაინის ძირითადი პრინციპების გაგება
  • კოდის სისუფთავის და სტრუქტურის შენარჩუნება
  • მრავალგვერდიანი ვებსაიტების შექმნა და ნავიგაცია

რატომ არის ეს მნიშვნელოვანი:

ყველაფერი, რასაც მომხმარებელი ვებსაიტზე ხედავს და იყენებს, შექმნილია Front-end დეველოპმენტის დახმარებით. პირადი ბლოგის, პორტფოლიოს ან სერიოზული ვებსერვისის შექმნისას — Front-end ცოდნა აუცილებელია ნებისმიერი დამწყები დეველოპერისთვის.
